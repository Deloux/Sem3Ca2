Documentation Project

Test strategies

We have used test driven development in the sense of test first design

Code design

The facade pattern used to implement jpa classes as fronts for the entities, and a restful design of the API

Inheritance strategy

We are using a joined inheritance strategy, since it seems the most logical compared to our business model
